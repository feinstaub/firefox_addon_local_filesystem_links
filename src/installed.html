Local Filesystem Links Extension

Important! Next steps required for the extension to work

  1. Download and install the native application for your system.
    Windows Linux (not ready yet)
  2. Check and modify whitelist settings by clicking at the addon icon in then addon bar. By default all pages are allowed but it's recommended to limit the pages where the addon can add the local link behaviour.
  3. Test the addon at your pages or at this Jsfiddle.

Additional information

Why do I need to install a program? Show answer

In previous versions of Firefox (before FF 57) it was possible to access the local filesystem with the browser API. That's not supported anymore and it is similar to other browsers (e.g. Chrome has also this restriction).

To make this extension possible we need a way to interact with the file system again. This is possible with a native application that can communicate with the browser with an API. It's called native messaging. The application will use the system file explorer (e.g. on Windows - it's Windows Explorer) to open the application in your linked application.
